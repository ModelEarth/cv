# About Resume Parser

Parses from a PDF file and converts into structured JSON for styling in HTML with selected themes. [Sample](#LorenHeyns)

### Features
- **PDF.js** extracts resume text data from the PDF file set in defaultPDF in your index.html page.
- Structured extracted text into JSON fields (name, email, skills, education, etc.).
- **JSON Resume themes** with live preview switching.
- Integrating **Gravatar** for automatic bio photo display (if no photo uploaded).

Developed by [Loren Heyns](#LorenHeyns), [Shanmuga Kannan](#ShanmugaPriyaKannan) and [Satvik Praveen](#SatvikPraveen)<!-- extract--> using [PDF.js](https://mozilla.github.io/pdf.js/getting_started/) and enhanced [JSON Resume Themes](https://jsonresume.org/themes)
